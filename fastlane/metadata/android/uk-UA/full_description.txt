Клієнтський застосунок для баз даних KeePass.

Ця програма призначена для мого особистого користування. Вона може містити деякі помилки, тому перед використанням зробіть резервну копію.

Функції програми:

Синхронізація:
 - Синхронізація з сервером WebDav
 - Синхронізація з репозиторієм Git (тільки через HTTPS, протокол SSH недоступний)

Управління базами даних:
 - Створення нових баз даних
 - Редагування існуючих баз даних
 - Управління історією внесень
 - Управління вкладеннями
 - Динамічні шаблони (сумісні з іншими додатками для Android: KeePassDX, keepass2android)

Розширені інструменти роботи з базами даних:
 - Вбудований переглядач змін у базі даних
 - Підтримка TOTP (одноразовий пароль на основі часу) та HOTP (одноразовий пароль на основі HMAC) для двофакторної автентифікації (2FA)

Функції безпеки:
 - Розблокування за допомогою пароля або файлу ключа
 - Підтримка біометричного розблокування (відбиток пальця)

Сумісність файлів:
 - Підтримка файлів .kdbx до версії 4.1

Зручність для користувача:
 - Автозаповнення для Android >= 8.0
 - Темна тема (керується системними налаштуваннями)
 - Нечіткий пошук для легкого знаходження записів
